Apollo Passive Seismic Experiment Expanded Event Catalog Bundle
Renee C. Weber

This bundle contains the following collections:

data
- The data collection contains the moonquake catalog csv files.

document 
- The document collection contains a description of the moonquake catalog bundle and lunar seismology background information as two PDF/A documents.

Each document and data product in this bundle is described by a PDS4 label in a separate file with the same name but with the extension XML. PDS4 labels are XML (eXtended Markup Language) files that conform to the PDS4 Information Model XML schema at https://pds.nasa.gov/pds4/schema/released/. They are best viewed in an XML-aware text editor, and may be used by software that can manipulate XML documents.

See these sources for more information.

- For information about the Apollo Seismic Event Catalog, see the document collection in this bundle. 
- For information about PDS4, see https://pds.nasa.gov/pds4/about/.
